# @draht/infra

SST v4 infrastructure for the Draht platform.

## Resources

- AWS Lambda functions
- API Gateway endpoints
- DynamoDB tables

## Usage

This package is deployed using [SST](https://sst.dev). See the `sst.config.ts` for resource definitions.

⚠️ **Do not run `sst deploy` manually** — deployments are managed via CI/CD.
